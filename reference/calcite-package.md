# calcite: Bindings to the Calcite Design System 'JavaScript' Component Library

Provides access to the 'Calcite Design System' 'javascript' components
via integration with the 'htmltools' and 'shiny' packages. Pre-built and
interactive components can be used to generate either static html or
interactive web applications. Learn more about the 'Calcite Design
System' at <https://developers.arcgis.com/calcite-design-system/>.
